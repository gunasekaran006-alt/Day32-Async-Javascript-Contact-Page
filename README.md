# 👤 User Profiles Dashboard - FakeStoreAPI Integration

A modern, responsive User Directory web application that fetches real-time data from an external API and displays it using a clean, card-based UI.

## ✨ Features
- **Live Data Fetching:** Integrates with `https://fakestoreapi.com` using Async/Await.
- **Dynamic UI:** Generates user cards on the fly with custom initials-based avatars.
- **Modern Design:** Built with Bootstrap 5 featuring glassmorphism elements, shadows, and color-coded contact badges.
- **Fully Responsive:** Optimized for Mobile, Tablet, and Desktop views using Bootstrap's Grid system.
- **Error Handling:** Includes loading spinners and user-friendly error messages for network failures.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3
- **Framework:** Bootstrap 5.3 (via CDN)
- **Icons:** Bootstrap Icons
- **Scripting:** Vanilla JavaScript (ES6+)
